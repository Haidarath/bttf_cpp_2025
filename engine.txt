Serialize all entities and their components to JSON format
Load scenes from JSON files, reconstructing entities and components
Must handle ALL component types you implemented in your engine (not just a
subset)
Human-readable JSON output format
Use nlohmann JSON library for C++ (https://github.com/nlohmann/json)

Specifications_
Test Program
You must write a separate small test program that:
• Loads a scene from a JSON file
• Runs the ECS system (executes your systems on the loaded entities)
• Prints relevant component values to console to show the ECS is working
You will demonstrate by: running the test program, manually editing the JSON file,
relaunching, and showing the ECS correctly processes the modified data.
